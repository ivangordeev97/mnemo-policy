# mnemo-policy
Политика конфиденциальности приложения МНЕМО
Настоящая политика конфиденциальности описывает, как приложение «МНЕМО» обрабатывает данные пользователей.

Сбор данных: Приложение является полностью оффлайновым. Мы не собираем, не передаем и не храним ваши данные (текстовые записи, фото, аудио) на внешних серверах.

Хранение: Все данные хранятся исключительно в локальной базе данных на вашем устройстве.

Разрешения: Приложение запрашивает доступ к микрофону (для распознавания речи) и камере (для фотофиксации вещей). Эти данные используются только внутри приложения.

Удаление: Вы можете в любой момент удалить все данные, удалив само приложение или записи в архиве.

Контакты: По вопросам политики пишите на [gordeev97@gmail.com].
